# Pagina 2 - Contexto del problema

## Que poner en la diapositiva

**Titulo:**

```text
Contexto: la ciberseguridad ya es un problema de datos
```

**Texto breve:**

```text
Las organizaciones generan miles de eventos de seguridad.
Los datos estan dispersos en incidentes, logs, alertas y fuentes historicas.
Sin integracion, la respuesta es lenta y reactiva.
Big Data permite correlacionar, analizar y priorizar riesgos.
```

## Imagen o icono sugerido

Usar una imagen con esta idea visual:

```text
Ataques -> logs dispersos -> analista saturado -> Big Data
```

Iconos recomendados:

- Alerta roja o incidente.
- Servidor/logs.
- Analista SOC.
- Base de datos.
- Lupa o grafico analitico.

## Que explicar oralmente

> En ciberseguridad no basta con tener logs o archivos CSV. El valor aparece cuando podemos unir esas fuentes y encontrar patrones. El problema es que cada fuente tiene su propio formato, nivel de detalle y tiempo. Por eso necesitamos una arquitectura que integre y procese datos a escala.

## Mensaje clave

```text
El problema no es solo tener datos; el problema es integrarlos y convertirlos en inteligencia accionable.
```
